# benchmark_plan_technical_drawing_script.py Acceptance Criteria

Use this reference when creating or reviewing the benchmark technical-drawing companion script.

## Role of the File

This script is the orthographic drawing companion for the benchmark plan.

## Acceptance Criteria

- The script matches the same geometry and labels as the evidence script.
- The script remains grounded in the benchmark contract.
- The script does not invent extra geometry, labels, or dimensions.
- The script is suitable for `preview_drawing()` before submission.
