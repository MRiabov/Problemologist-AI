# benchmark_plan_evidence_script.py Acceptance Criteria

Use this reference when creating or reviewing the benchmark plan evidence script.

## Role of the File

This script is the inspectable source for the benchmark geometry evidence scene.

## Acceptance Criteria

- The script is legible and grounded in the benchmark contract.
- The script preserves inventory labels and quantities exactly.
- The script does not drift into solution-side geometry.
- The script is suitable for visual inspection by downstream roles.
