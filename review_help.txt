                              
 Usage: spec-kitty agent      
 workflow review [OPTIONS]    
 [WP_ID]                      
                              
 Display work package prompt  
 with review instructions.    
                              
 This command outputs the     
 full work package prompt     
 (including any review        
 feedback from previous       
 reviews) so agents can       
 review the implementation.   
                              
 Automatically moves WP from  
 for_review to doing lane     
 (requires --agent to track   
 who is reviewing).           
                              
 Examples:                    
     spec-kitty agent         
 workflow review WP01 --agent 
 claude                       
     spec-kitty agent         
 workflow review wp02 --agent 
 codex                        
     spec-kitty agent         
 workflow review --agent      
 gemini  # auto-detects first 
 for_review WP
